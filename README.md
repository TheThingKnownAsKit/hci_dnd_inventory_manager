# hci_dnd_inventory_manager
## Setup Repository
1. Clone the repository via `git clone https://github.com/TheThingKnownAsKit/hci_dnd_inventory_manager.git`
2. Open the repository folder in VSCode
3. Install prerequisites

    3a. Ensure that Python 3.12+ is installed
   
    3b. Ensure that the Python VSCode Extension (ms-python.python) is installed
   
    3c. Ensure that Node.js is installed
   
5. Create a Python Virtual Environment
    4a. Press Ctrl + Shift + P
   
    4b. Enter "Python: Create Environment..."
   
    4c. Select Venv
   
    4d. Select either main Python path it shouldn't matter
   
    4e. Select requirements.txt as dependencies to install

## Running the Project
Note that this project really doesn't like to work on any browser that isn't Chromium, so Google Chrome is recommended.

1. Source the Python virtual environment
2. Run `reflex run` in the terminal
3. Wait for it to compile and visit the website at http://localhost:3000/
